Bus Tracking System

A smart real-time Bus Tracking System developed using modern web technologies to help users track bus locations efficiently on an interactive map interface.

This project integrates map visualization, mobile support, and real-time location tracking concepts to improve transportation monitoring and user convenience.

Project Objective

The main objective of this project is to create a smart transportation tracking solution that allows users to monitor bus movement and improve travel management using map-based visualization.

Installation & Setup
Step 1: Clone Repository
git clone https://github.com/your-username/bus-tracking-system.git
Step 2: Open Project Folder
cd bus-tracking-system
Step 3: Install Dependencies
npm install
Step 4: Run Project
npm start
Mobile Build (Android)

To run the project on Android using Capacitor:

npx cap add android
npx cap sync
npx cap open android
